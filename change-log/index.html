Version 1.4.0

  • Improved card gesture reliability

  • Gesture support for switching between upcoming and completed reminders
  • Minor UI improvements
  • Chinese language support

Version 1.3.0

  • Added four new default images. Squidy we hardly knew ye!

  • Updated syncing and error handling
  • Added the version and build numbers to the Settings app
  • Misc bug fixes

Version 1.2.0
